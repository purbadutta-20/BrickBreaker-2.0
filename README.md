# 🎮 Brick Breaker Pro

A professional Brick Breaker arcade game built using Python and Pygame with modular architecture and advanced gameplay features.

---

## 🚀 Features

- Multiple Levels
- Difficulty Modes (Easy, Medium, Hard)
- Combo Multiplier System
- Particle Effects
- Powerups
- Pause System
- Game Over Screen
- High Score Saving
- Sound Effects and Music

---

## 🛠 Tech Stack

- Python 3.10+
- Pygame
- Object-Oriented Programming

---

## 📦 Installation

Clone repository:
        
git clone : git remote add origin https://github.com/purbadutta-20/Break-Breaker-Pro.git

Go to folder:

cd brick-breaker-pro

Install dependencies:

pip install -r requirements.txt

Run the game:

python main.py

---

## 🎮 Controls

LEFT / RIGHT → Move paddle  
SPACE → Start game  
P → Pause  
R → Restart  
1 / 2 / 3 → Select difficulty  

---

## 📁 Project Structure

entities/ → game objects  
ui/ → menu and HUD  
utils/ → sound and storage  
assets/ → sounds  
game.py → main game logic  
main.py → entry point
